# 🏫 國小圖書管理系統 - 使用者測試案例 (User Test Cases)

請依照下列步驟進行系統功能驗收。

## 🛠️ 前置準備
1. 確保終端機位於 `school_library` 目錄。
2. 執行初始化指令以產生測試資料：
   ```bash
   python3 seed_data.py
   ```
3. 啟動伺服器：
   ```bash
   python3 -m uvicorn main:app --reload
   ```
4. 打開瀏覽器進入：`http://127.0.0.1:8000`

---

## 🧪 測試案例清單

### 1. 儀表板與統計 (Dashboard)
- [ ] **檢查首頁數據**：
    - 確認「總藏書量」是否為 11 本以上。
    - 確認「逾期未還」是否顯示紅色數字 (預設應為 1)。
- [ ] **查看統計圖表**：
    - 點擊導覽列「統計分析」。
    - 確認是否出現「每日借書量」的折線圖。

### 2. 書籍管理 (Book Management)
- [ ] **搜尋書籍**：
    - 在書籍頁面輸入 "哈利波特" 或 "羅琳"。
    - 確認搜尋結果正確顯示。
- [ ] **分類篩選**：
    - 下拉選單選擇 "科普"。
    - 確認列表只顯示科普類書籍。
- [ ] **新增書籍 (Google API 測試)**：
    - 點擊「➕ 新增」。
    - 在 ISBN 欄位輸入 `9789573317241`。
    - 點擊「🔍 抓取資料」或是按下 Enter。
    - **預期結果**：書名、作者、分類欄位應自動填入。
    - 點擊「確認新增」。
    - **預期結果**：列表出現新書，且複本編號自動 +1。
- [ ] **電子書連結**：
    - 找到《小王子》或《哈利波特》。
    - 點擊紫色的「🌐 線上閱讀」按鈕，確認是否在新分頁開啟 Google Books。

### 3. 學生與借書證 (Student & Cards)
- [ ] **查看學生名單**：
    - 點擊導覽列「學生管理」。
    - 確認出現測試資料中的 10 位學生。
- [ ] **借書證產生**：
    - 點擊「張小明」右側的「🪪 借書證」。
    - 確認畫面顯示學校名稱、學生資料與條碼。
    - 點擊「🖨️ 立即列印」，確認列印預覽中**沒有**導覽列與按鈕 (只顯示卡片)。
- [ ] **借閱歷史**：
    - 點擊「張小明」的「📜 歷史」。
    - 確認看到一筆「逾期」的借閱紀錄。

### 4. 借閱與歸還 (Loans & Returns)
- [ ] **借書操作 (正常)**：
    - 記下任意一本 "在館中" 的書籍 ID (例如 #5) 與學生學號 (例如 S003)。
    - 在首頁「快速登記借書」輸入上述 ID。
    - **預期結果**：跳轉回首頁，該書狀態變為 "已借出"。
- [ ] **借書上限測試**：
    - 嘗試為同一位學生連續借閱 6 本書。
    - **預期結果**：第 6 本時首頁出現紅色錯誤提示：「🚫 借書失敗：該學生已達到借書上限」。
- [ ] **歸還操作**：
    - 點擊導覽列「借閱紀錄」。
    - 找到標示「⚠️ 已逾期」的紀錄。
    - 點擊綠色「📗 歸還」按鈕。
    - **預期結果**：紀錄從列表中消失 (或移至歷史紀錄)，書籍狀態變回 "在館中"。

### 5. 智慧掃描 (Scanner)
- [ ] **掃描器測試**：
    - 點擊導覽列「📷 掃描器」。
    - 同意瀏覽器使用相機權限。
    - 拿一本書的 ISBN 條碼對準鏡頭。
    - **預期結果**：系統偵測到 ISBN，並顯示「查詢此書」按鈕。
    - (若無條碼可掃，可嘗試用手機螢幕顯示 ISBN 條碼給鏡頭看)

### 6. 資料匯入匯出 (Import/Export)
- [ ] **匯出 CSV**：
    - 在書籍頁面點擊「📤 匯出」。
    - 下載 `books.csv`，用 Excel 打開，確認中文顯示正常 (無亂碼)。
- [ ] **匯入 CSV**：
    - 修改剛才下載的 CSV，新增一行資料。
    - 在書籍頁面選擇該檔案並點擊「📥 匯入」。
    - **預期結果**：書籍列表中出現剛才新增的資料。

---
**測試完成！** 🎉
若所有項目皆打勾，代表系統功能運作正常且穩定。
